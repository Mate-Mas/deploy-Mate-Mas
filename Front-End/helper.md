# Comandos utiles

Antes de empezar a trabajar desde la carpeta raíz 

`git pull`

`pnpm install:all`

Antes de subir cambios, deberías bajar los últimos cambios

Traer los cambios

`git pull`

Agregar archivos (El "." agrega TODOS los archivos modificados)

`git add .`

ó

`git add + (nombre archivo)` # ejemplo: `git add app.js`

Subir los cambios 

`git push origin (nombre de la rama)` # ejemplo: `git push origin lisandro`
